**Role:** Expert UI/UX Designer and Frontend Developer

**Task:** Generate a high-fidelity, interactive prototype for a Personal Homepage and Blog using HTML, Tailwind CSS, and vanilla JavaScript (or React if preferred, but keep it suitable for an AI generator like Stitch).

**Design Style:** "Modern Dark" aesthetic combined with a "Geek" vibe.
*   **Theme:** Dark mode by default. Deep, rich blacks and very dark grays for backgrounds.
*   **Typography:** Clean, modern Sans-serif fonts. Use high contrast for readability.
*   **Vibe:** Sleek, professional, minimalist, and "Geeky". Subtly evoke the "Char Aznable" theme from Mobile Suit Gundam (mechanical, precise, red comet).
*   **UI Elements:** Use subtle borders, soft glows or neon-like accents for interactivity.

**Strict Constraint:** Do NOT include any images, photos, or character avatars in the prototype. Use typography, layout, CSS styling, icons (like SVG), and geometric shapes to build the visual hierarchy.

*   **Theme Vibe:** Char Aznable's iconic color scheme.
*   **Main Colors:** Deep Black (backgrounds), Red, and Gold.
*   **Accent Color:** Bright Pink-Red.
*   **Text Colors:** Red to Bright Pink-Red (use gradients for primary text/headings to create a glowing geek effect).
